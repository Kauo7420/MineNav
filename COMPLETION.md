## 🛠️ 技术规格

### 前端技术
- **HTML5** - 语义标记
- **CSS3** - 现代样式和布局
- **Vanilla JavaScript** - 零依赖

### 设计系统
- **颜色** - Modrinth (绿 #1bd96a) / SpigotMC (橙 #f6a821)
- **排版** - Inter (Google Fonts)
- **间距** - 8px 网格系统
- **动画** - 流畅的 300ms 过渡

### API 集成
- **Modrinth** - v2 API (`api.modrinth.com/v2`)
- **SpigotMC** - Spiget API (`api.spiget.org/v2`)
- **缓存策略** - 5 分钟缓存
- **错误处理** - 优雅降级

## 🔄 工作流程

### 用户流程
```
启动应用
    ↓
检测系统主题 → 应用主题
    ↓
加载 API 统计
    ↓
显示插件列表
    ↓
用户操作 (搜索/过滤/排序)
    ↓
实时更新内容
    ↓
点击插件 → 显示详情
    ↓
访问官方网站
    ↓
返回浏览
```

## 💾 数据流

```
用户输入
    ↓
验证和处理
    ↓
检查缓存
    ↓
缓存命中？
├─ 是 → 返回缓存数据
└─ 否 → 调用 API
    ↓
格式化数据
    ↓
更新 DOM
    ↓
应用动画
    ↓
显示结果
```
