# PluginMarket 快速开始指南

欢迎使用 **PluginMarket** - Minecraft 插件导航平台！

## 🚀 立即开始

### 方式 1：直接打开（推荐）
1. 在文件浏览器中找到项目文件夹
2. 双击 `index.html` 文件
3. 浏览器会自动打开网站

### 方式 2：使用本地服务器（更好的兼容性）

#### Python 3
```bash
cd /path/to/MineNav
python -m http.server 8000
```
然后访问：`http://localhost:8000`

#### Node.js
```bash
cd /path/to/MineNav
npx http-server
```
然后访问：`http://localhost:8080`

#### Live Server (VS Code)
1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## ✨ 核心功能

### 1. 搜索插件
- **方式**: 在顶部搜索框输入插件名称或关键字
- **快捷键**: 按 `/` 在任何地方打开搜索框
- **跨平台**: 默认搜索 Modrinth 和 SpigotMC 的所有插件

### 2. 选择平台
```
┌─────────────────────────┐
│ [全部] [Modrinth] [Spiget] │
└─────────────────────────┘
```
- **全部** - 显示两个平台的插件
- **Modrinth** - 仅显示 Modrinth 插件（绿色）
- **SpigotMC** - 仅显示 SpigotMC 插件（橙色）

### 3. 按分类浏览
水平滚动导航条包含以下分类：
- 全部
- 管理工具 - 服务器管理插件
- 游戏内容 - 游戏机制扩展
- 世界管理 - 世界生成和管理
- 经济系统 - 经济类插件
- 聊天系统 - 聊天和社交
- 实用工具 - 服务器工具
- 开发工具 - 库和API
- 杂项 - 其他插件

### 4. 排序和筛选
- **排序选项**:
  - 按下载量排序
  - 按最新发布排序
  - 按最近更新排序
  - 按关注数排序

- **视图模式**:
  - 网格视图 (👁) - 卡片网格展示
  - 列表视图 (☰) - 紧凑列表展示

### 5. 查看插件详情
1. 点击任意插件卡片
2. 在弹出的模态框中查看：
   - 完整描述
   - 下载量、关注数、更新时间
   - 支持的版本和加载器
   - 链接到官方网站

### 6. 主题切换
- 点击右上角的 **☀️** (浅色) 或 **🌙** (深色) 按钮
- 偏好设置会自动保存
- 首次访问时自动检测系统偏好

## 🌐 API 数据

### 数据来源
- **Modrinth** - `api.modrinth.com/v2`
- **SpigotMC** - `api.spiget.org/v2`

### 更新频率
- 数据每 5 分钟缓存一次

## 🔧 自定义和配置

### 修改每页显示数量
编辑 `scripts/config.js`:
```javascript
CONFIG.ITEMS_PER_PAGE = 12; // 改为其他数字，如 20
```

### 修改搜索防抖延迟
```javascript
CONFIG.SEARCH_DEBOUNCE = 300; // 毫秒，越小反应越快
```

### 修改颜色主题
编辑 `styles/variables.css`:
```css
--color-primary-modrinth: #1bd96a;  /* 绿色 */
--color-primary-spigot: #f6a821;    /* 橙色 */
```

## 🐛 常见问题

### Q: 页面加载缓慢
**A**: 这可能是因为：
1. 网络连接较慢 - 等待几秒钟
2. API 响应慢 - 刷新页面重试
3. 浏览器缓存满 - 清除浏览器缓存

### Q: 搜索不返回结果
**A**: 请尝试：
1. 检查网络连接
2. 使用不同的搜索词
3. 检查平台是否选择正确
4. 刷新页面
5. 检查浏览器控制台的错误信息 (F12)

### Q: 为什么某些插件在手机上显示不完全？
**A**: 这是正常的。长文本会被截断以节省空间。点击插件卡片查看完整信息。

### Q: 主题不会改变
**A**: 
1. 检查 JavaScript 是否已启用
2. 尝试刷新页面
3. 清除浏览器缓存
4. 手动删除 localStorage (高级)

### Q: SpigotMC 插件无法加载
**A**: SpigotMC API 可能有 CORS 限制。这是已知问题，应用会优雅地处理。
